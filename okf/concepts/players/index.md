# Players

Player concepts. Each focuses on identity, canonical resource, relationships, and agent
guidance. Volatile stats are sourced and dated; for current values, follow the canonical
page. The full IPL 2026 player set is 256; this v0.1 bundle curates high-signal players.

## Concepts

- [Virat Kohli](virat-kohli.md) — RCB, batter
- [Jasprit Bumrah](jasprit-bumrah.md) — MI, bowler
- [Vaibhav Suryavanshi](vaibhav-suryavanshi.md) — RR, batter

## What Agents Should Know

A player concept is an identity anchor. Numbers attached to it are snapshots with a
`computed_at` date and sample size — never treat them as live. Pair with the relevant
[metric](../../metrics/index.md) definition and state the season. See
[When not to rank a player](../../examples/when-not-to-rank-a-player.md).
